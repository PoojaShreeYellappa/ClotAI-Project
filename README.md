# 🩸 ClotAI: Precision Blood Matching System

### 🚀 Overview
ClotAI is a data-driven platform built to solve the delay in finding blood donors during emergencies in India. It treats blood units like critical inventory, matching them to patients based on location and compatibility.

### 📊 Business Logic (BA Focus)
*   **Logistics:** I used the Haversine formula to calculate the exact distance between donors and hospitals, optimizing for the fastest response.
*   **Automated Verification:** Built a logic-gate system based on NBTC India guidelines to automatically check donor eligibility (Age, Weight, etc.).
*   **Inventory Tracking:** Simulated live stock levels for major Bengaluru hospitals like Manipal and Narayana Health.

### 🛠 Tech Stack
*   **Frontend:** React.js
*   **Data Logic:** JavaScript (Arrays for blood compatibility and Math for distance filtering).
